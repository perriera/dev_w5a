What is ng-indent?

ng-indent

FreeFormJS brings a whole new developer experience when editing Angular 2 code in your favorite editor, whether it be Visual Code Studio, WebStorm, Eclipse or even vi. Such that, by default, it will highlight Angular 2 code for you. Such that you can better see where your Angular 2 enhanced HTML tags are located. It makes for an incredibly enjoyable editing experience. Nothing like this has ever been seen in modern code editors before and it is expected to start a trend.

Why use ng-indent?

It makes life a lot easier when it comes to editing your Angular 2 enhanced HTML code.

ng-indent

Step by step:

  1. ng-indent is not applied as a custom attribute. It's a setting in the iniFile for the project. You don't have to do anything, unless you turn it off in the iniFile, it formats you code automatically.
  2. However, in order to select which HTML elements get selected for formatting with ng-indent, you simply supply that element with an id value.
  3. This allows you to easily select which fields are formatted thereby giving you a much easier editing experience for the Angular 2 developer.

Caution:

By default any HTML elements that have an id value will be formatted for possible use by Angular 2 custom attributes in mind. Event HTML elements that you are not intending to use for Angular 2. So assign id's only as necessary to keep you code looking sharp and pleasant to deal with.
