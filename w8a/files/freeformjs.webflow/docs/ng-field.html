What is ng-field?

ng-field

The ng-field is a pseudo custom attribute that allows you to add fields in your Webflow project that will be supported in the traditional Angular 2 fashion. At the moment Webflow will not allow you to use the { { and } } braces in your text. The ng-field technique solves this issue.

ng-field

Step by step:

  1. All you have to do is identify where in your web page (inside the Webflow editor) where you want Angular 2 style fields to show up and use the ( ( and ) ) symbols instead.
  2. During the conversion process FreeFormJS will change the ( ( and ) ) brackets into the { { and } } braces for you.
  3. There is no need to define an actual ng-field for the custom attributes as this point in time and doing so would only add extra complexity to your editing process.
  4. Also, you do not have to set the id value for this technique to work.

Caution:

Also note that at the time of this writing that using either double braces or double brackets inside the Webflow editor will not convert at all during the conversion process. The spaces between { {, } }, ( ( and ) ) are intentional here to show you the symbols we are referring to. When you add brackets to in Webflow webpage be sure to have no space in between in order for this to work properly.
