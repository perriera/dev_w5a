What is ng-router?

ng-router

By default FreeFormJS performs a simple carbon copy of your index page layout to each of your other HTML pages. Usually, you want to make use of some the advantages of using Angular 2 which includes keeping part of the index page static and using the other HTML pages as parts of the main index page. To make use of Angular 2's route mechanism in this manner, assign ng-router to some part of the index.html page:

ng-router

Step by step:

  1. The first thing you need to do is identify where in your index page you'd like to section off as static and what part to be sectioned off as dynamic. That is, where all the other pages will display as you click menu items.
  2. Inside the web page select the leading <div> block for that area of the web page you'd like to make into a component.
  3. While the <div> is selected, click on the icon to the right of the paint brush you see to the top right of the page, it looks like a star. In that section of the Webflow editor you will see the Custom Attributes section where you click the plus sign to add a custom attribute.
  4. Click on the plus sign and add ng-router and assign it to "home" as the value.
  5. All you have to do now is Code Export your project and pass it through FreeFormJS for conversion.
  6. FreeFormJS will do five things:
  • The first thing it'll do is completely remove all HTML code from index.html and in it's place put an Angular 2 <app-root> tag.
  • The second thing it'll do is create a home.component.ts for you and place all the default home page values you have in your index.html there.
  • The third thing it'll do is take all the static portion of the index.html file and place it inside the app.component.ts with a special addition. It'll place the <router-outlet> value inside where the original index content used to be.
  • The fourth thing it'll do is sort out all the other pages in your Webflow project and place references to them inside app-routing.module.ts. This allows Angular 2's standard routing mechanism to function as intended.
  • The fifth thing it'll do take all your images and place them inside the assets/ directory as well as take all your css style sheets and merge them into the styles.css file.
You will be able to disable defaults feature of FreeFormJS through the use of a iniFile we can provide you if you like. But this is the default feature of FreeFormJS can be either customized through the use of an iniFile which will be available on this site shortly.
